基本介绍​

通用时间管理模块，封装了常用的时间/日期相关的方法，作为标准库 time 的功能性扩展，提供了更多的功能特性。支持自定义的日期格式化语法，格式化语法灵感来源于 PHP 的 date 函数语法 ( http://php.net/manual/zh/function.date.php )。

tip

gtime 的时间格式语法对于熟悉 PHP 的开发者来说非常友好。

使用方式：

import "github.com/gogf/gf/v2/os/gtime"

接口文档：

https://pkg.go.dev/github.com/gogf/gf/v2/os/gtime
